## 核心发现

### 优势（护城河）
1. **零售银行品牌认知强**："零售之王"深入人心
2. **财富管理先发优势**：私行业务领先，AUM超3.5万亿元
3. **风险管理能力出色**：不良率0.95%，拨备覆盖率450%
4. **客户转换成本高**：账户体系复杂，代发工资捆绑
5. **财务质量优异**：ROE 13.44%，净利润率44.8%

### 风险与挑战
1. **房地产风险暴露**：房地产贷款占比约25.4%，不良率约5.28%
2. **增长失速**：净利润增速从2022年15.1%降至2025年1.2%
3. **净息差收窄**：从2.48%下降至2.03%，利率市场化压力
4. **金融科技颠覆**：蚂蚁、腾讯等跨界竞争
5. **宏观经济下行**：经济增速放缓，银行业顺周期承压
