## 研究方法

本研究采用**未上市公司多Agent并行研究框架**：

- **并行启动**：5位AI分析师同时独立研究，避免互相干扰
- **多源核验**：每个关键数据点至少2个独立来源交叉验证
- **置信度标注**：所有数据标注 🟢高 / 🟡中 / 🔴低
- **诚实留白**：信息缺失处明确标注"未公开"，不强行填充
- **双口径分离**：营收等关键数据明确区分管理口径 vs 第三方可比口径
- **对标驱动**：用22家上市可比公司的实际估值水位反推合理估值
